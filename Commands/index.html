This section is dedicated to a lovely list of commands

Prefix: ! (not change-able)

    PUBLIC COMMANDS

  • help - shows this list
  • info - shows bot info
  • ched - summons a random emoji
  • invite - link to invite the bot
  • avatar - shows the users avatar (profile image)
  • p - parcel
  • explain {what to explain args} - explains something (info from Wikimedia)

    • ADMIN COMMANDS

  • ping - shows the server-to-chat ping in ms
  • lol - become funi admin
  • clear {number args} - clears a certain amount of messages (100 message cap)
  • warn {user arg} {reason arg} - warns a user for some reason
  • serverinvite - tired of manually making invites? SYSbot now does that for you!
  • ban {user arg} - bans a user... where'd I leave that ban hammer?

    • PRIVATE COMMANDS

  • debug - only Admins on the Official Support Server can use this one
  • spam - only the creator of the bot can use this one
  • todo - only the Dev Team Members can use this one
